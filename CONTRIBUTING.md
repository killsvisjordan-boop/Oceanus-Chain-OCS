# Contributing to Oceanus Chain 🌊

Thank you for your interest in contributing to Oceanus Chain! We welcome developers, researchers, and creators to help us build the first AI Layer 1 ecosystem on Ethereum.

## How Can I Contribute?

### 1. Reporting Bugs
* Check the Issues tab to see if the bug has already been reported.
* If not, open a new issue with a clear title, description, and steps to reproduce the bug.

### 2. Submitting Code Changes
1. Fork the repository.
2. Create a new branch for your feature (`git checkout -b feature/amazing-feature`).
3. Commit your changes with clear messages (`git commit -m 'Add some amazing feature'`).
4. Push to the branch (`git push origin feature/amazing-feature`).
5. Open a Pull Request (PR) against the `main` branch.

### 3. Coding Standards
* Ensure your code is clean, well-commented, and follows standard Solidity/TypeScript guidelines where applicable.
* Make sure all automated tests pass before submitting your PR.

We appreciate your support in making Oceanus deeper and stronger!
