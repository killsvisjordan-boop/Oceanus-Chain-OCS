# Security Policy 🛡️

## Vulnerability Reporting
We take the security of Oceanus Chain very seriously. If you discover a security vulnerability within this repository or the wider ecosystem, **please do not open a public GitHub issue.** Instead, report it privately to ensure the safety of our smart contracts and assets.

### How to Report a Vulnerability
Please send an email to: **security@oceanuschain.xyz** *(or contact team developers via official secure channels)* with:
* A detailed description of the vulnerability.
* Steps or a proof-of-concept (PoC) to reproduce the issue.
* The potential impact of the vulnerability.

## Disclosure Process
Once a report is received:
1. We will acknowledge receipt of your report within 48 hours.
2. We will investigate the issue and coordinate a fix.
3. We will keep you updated throughout the process.

Thank you for helping keep the Oceanus ecosystem secure!
